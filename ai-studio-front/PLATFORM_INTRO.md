# 面向训推一体与大小模型协同的AI Studio人工智能开发平台

## 1. 题目
面向训推一体与大小模型协同的AI Studio人工智能开发平台

## 2. 开发目的
构建训推一体化平台，融合大小模型协同能力，降低开发门槛，实现从数据到服务的全链路高效AI应用落地。

## 3. 主要介绍
本项目是一款集数据管理、模型开发、训练编排、服务部署于一体的企业级人工智能开发平台。平台旨在降低AI应用落地门槛，通过“训推一体化”架构，实现从数据标注到模型服务的全链路打通。特别针对当前AI发展趋势，创新性地支持“大小模型协同”工作流，既支持传统视觉/预测类小模型的快速训练，也融合了大模型的微调与应用能力，为科研与产业应用提供高效、灵活的AI基础设施。

## 4. 研究内容
1.  **全生命周期AI开发流程研究**：探索并实现涵盖数据采集、清洗标注、模型构建、训练调优、评估验证及服务部署的端到端标准化流程。
2.  **异构算力资源调度技术**：研究基于容器化的CPU/GPU异构资源池化管理与动态调度策略，提高算力利用率。
3.  **可视化模型编排机制**：设计基于DAG（有向无环图）的低代码拖拽式建模工具，降低算法开发复杂度。
4.  **大小模型协同进化机制**：研究大模型辅助小模型标注（智能标注）、小模型反馈大模型微调的协同范式，以及不同规模模型在同一平台下的统一纳管。

## 5. 关键技术
1.  **前端可视化技术**：基于Vue 3 + Vite架构，采用ECharts实现训练过程与集群监控的实时可视化，利用Ant Design/Element Plus构建响应式交互界面。
2.  **微服务与容器化**：采用前后端分离架构，后端对接Kubernetes集群，实现训练任务与推理服务的容器化隔离与弹性伸缩。
3.  **模型流水线技术**：实现了可配置的Pipeline引擎，支持数据处理、特征工程、模型训练等节点的灵活编排与参数透传。
4.  **多框架兼容适配**：兼容PyTorch, TensorFlow等主流深度学习框架，并适配多种国产化算力环境。

## 6. 主要功能
1.  **工作台总览**：实时展示数据集、模型、训练任务统计，以及集群节点（CPU/GPU/内存）负载监控。
2.  **数据中心**：支持多模态数据管理，提供数据集分类统计、智能辅助标注及标注进度可视化。
3.  **模型中心**：构建企业级模型仓库，支持按应用场景（视觉、NLP等）、任务类型及框架筛选，提供模型版本管理与多维评估。
4.  **模型产线（开发）**：提供可视化编排画布，支持创建训练任务、配置算法参数、实时查看训练日志及评估指标。
5.  **服务部署**：支持模型一键部署为在线API服务，提供服务调用监控、日志查询及启停管理。
6.  **系统管理**：完善的RBAC权限体系，包含用户、角色、部门及菜单管理，保障平台安全。

## 7. 特点与创新
1.  **训推一体化设计**：打破训练与推理的壁垒，训练产出的模型可直接无缝推送到服务部署模块，大幅缩短模型上线周期。
2.  **大小模型协同赋能**：平台不仅支持常规CV/NLP模型训练，还集成了大模型能力，利用大模型强大的泛化能力辅助数据标注（预标注），提升小模型生产效率。
3.  **低代码可视化编排**：将复杂的算法代码封装为可视化组件，用户通过简单的拖拽连线即可构建复杂的训练流水线，极大降低了AI开发门槛。
4.  **精细化资源管控**：提供细粒度的集群监控与任务排队机制，确保高并发下的系统稳定性与资源公平性。
