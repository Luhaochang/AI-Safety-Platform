# 3D Models Directory

Place `.glb` or `.gltf` 3D model files here for the Digital Twin scene.

## Recommended Free Model Sources
- [Sketchfab](https://sketchfab.com) - Search "transformer electrical", "server rack", "power substation"
- [Turbosquid](https://www.turbosquid.com/Search/Index.cfm?keyword=&max_price=0) - Free section
- [Poly Haven](https://polyhaven.com/models) - CC0 licensed models

## Expected Model Files
- `transformer.glb` - Power transformer model
- `server_rack.glb` - Server/computing rack model
- `switchgear.glb` - Electrical switchgear cabinet
- `cooling_unit.glb` - Cooling tower/unit

Models will be automatically loaded if present. The scene uses geometric primitives as fallback.
