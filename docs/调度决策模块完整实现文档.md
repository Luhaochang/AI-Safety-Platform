## 模块概述

调度决策模块包含 4 个子功能：
1. **调度助手** - 基于 Qwen 大模型的智能对话助手
2. **知识库管理** - 电力调度规程、故障处置手册等知识文档管理
3. **孪生推演** - 电网故障仿真推演（N-1 校核、极端场景分析）
4. **决策日志** - 对话记录 + 推演记录的统一审计日志

export async function callOpenAI(query, historyMessages = [], model = 'gpt-4') {
    const apiKey = import.meta.env.VITE_OPENAI_API_KEY;
    if (!apiKey) return null;

    const messages = [
        { role: 'system', content: QWEN_SYSTEM_PROMPT },
        ...historyMessages.slice(-6),
        { role: 'user', content: query }

    const res = await axios.post('https://api.openai.com/v1/chat/completions', {
        model,
        messages,
        temperature: 0.7,
        max_tokens: 2000
    }, {
        headers: {
            'Authorization': `Bearer ${apiKey}`,
            'Content-Type': 'application/json'
        }
    });
    return res.data;
}

**发送消息：** `src/views/ragScheduler/chat.vue:60-125`

```javascript
const sendMessage = async () => {
    // 1. 获取或创建会话ID
    let convId = chatStore.currentConversationId;
    if (!convId) {
        convId = chatStore.createNewConversation();
    }

    // 2. 添加用户消息到 store
    const userMsg = { role: 'user', content: text, time: new Date().toLocaleTimeString() };
    chatStore.addMessage(convId, userMsg);

    // 3. 调用大模型 API
    if (state.useMockMode || !qwenConnected.value) {
        // Mock 模式
        const mockRes = await sendChatMessageMock({ query: text });
        assistantContent = mockRes.data.content;
    } else {
        // Qwen API（传入选中的模型名称）
        const qwenRes = await callQwenApi(text, currentMessages.value, state.selectedModel);
        assistantContent = qwenRes.choices[0].message.content;
    }

    // 4. 添加助手回复到 store
    const assistantMsg = {
        role: 'assistant',
        content: assistantContent,
        references: [],
        time: new Date().toLocaleTimeString()
    };
    chatStore.addMessage(convId, assistantMsg);
};
```

### Pinia Store 设计

**文件：** `src/store/modules/chat.js`

```javascript
export const useChatStore = defineStore('chat', {
    state: () => ({
        conversations: [
            {
                id: 'conv_1',
                title: '220kV线路故障处理',
                messages: [
                    { role: 'user', content: '...' },
                    { role: 'assistant', content: '...' }
                ],
                retrievedPassages: [
                    { title: '电力调度规程', content: '...', score: 0.92 }
                ],
                user: '调度员A',
                duration: '2.3s',
                createTime: '2024-03-18 20:30:15'
            }
        ],
        twinSimulations: [
            {
                id: 'twin_1',
                title: '220kV线路N-1故障推演',
                result: {
                    summary: '...',
                    affectedLines: [...],
                    recommendations: [...],
                    riskLevel: '低风险'
                },
                user: '系统运行员',
                duration: '3.2s',
                createTime: '2024-03-18 19:45:20'
            }
        ],
        currentConversationId: null
    }),
    
    getters: {
        // 当前对话
        currentConversation: (state) => {
            return state.conversations.find(c => c.id === state.currentConversationId);
        },
        
        // 合并决策日志
        decisionLogs: (state) => {
            const logs = [];
            state.conversations.forEach(conv => {
                logs.push({ type: 'chat', ...conv });
            });
            state.twinSimulations.forEach(sim => {
                logs.push({ type: 'twin', ...sim });
            });
            return logs.sort((a, b) => new Date(b.createTime) - new Date(a.createTime));
        }
    },
    
    actions: {
        // 创建新对话
        createNewConversation() {
            const newConv = {
                id: 'conv_' + Date.now(),
                title: '新对话',
                messages: [],
                user: '调度员',
                createTime: new Date().toLocaleString('zh-CN', { hour12: false })
            };
            this.conversations.unshift(newConv);
            this.currentConversationId = newConv.id;
            return newConv.id;
        },
        
        // 添加消息
        addMessage(convId, message) {
            const conv = this.conversations.find(c => c.id === convId);
            if (conv) {
                conv.messages.push(message);
                // 自动更新标题
                if (conv.messages.length === 2) {
                    conv.title = conv.messages[0].content.substring(0, 20) + '...';
                }
            }
        },
        
        // 添加推演记录
        addTwinSimulation(simulation) {
            this.twinSimulations.unshift(simulation);
        }
    }
});
```

export async function callQwenApi(query, historyMessages = [], modelName = 'qwen-plus') {
    const apiKey = import.meta.env.VITE_QWEN_API_KEY;
    if (!apiKey) return null;

    // 构建消息上下文
    const messages = [
        { role: 'system', content: QWEN_SYSTEM_PROMPT },
        ...historyMessages.slice(-6).map(m => ({ role: m.role, content: m.content })),
        { role: 'user', content: query }
    ];

    // 调用 DashScope API
    const res = await axios.post('/dashscope-api/compatible-mode/v1/chat/completions', {
        model: modelName,
        messages,
        temperature: 0.7,
        max_tokens: 2000
    }, {
        headers: {
            'Authorization': `Bearer ${apiKey}`,
            'Content-Type': 'application/json'
        },
        timeout: 30000
    });
    
    return res.data;
}

## 常见问题

### Q1: 为什么切换模型后没有生效？

**A:** 确保你点击了下拉菜单中的模型选项。可以在浏览器控制台查看 `state.selectedModel` 的值。

### Q2: Token 额度显示不准确？

**A:** 系统显示的是模拟数据。真实额度请登录 DashScope 控制台查看。

### Q3: 如何让 Mock 模式返回更真实的数据？

**A:** 编辑 `src/api/ragScheduler/index.js:92-110`，在 `responses` 数组中添加更多电力调度相关的回复模板。

### Q4: 推演结果能否保存到数据库？

**A:** 当前推演结果存储在 Pinia Store（内存中），刷新页面会丢失。如需持久化，可以：
1. 在 `src/api/ragScheduler/index.js` 添加 `saveTwinSimulation(data)` 函数
2. 调用后端 API 保存到数据库
3. 页面加载时从后端获取历史推演记录

### Q5: 如何调整 Qwen 的回复风格？

**A:** 修改 `src/api/ragScheduler/index.js:8-9` 的 `QWEN_SYSTEM_PROMPT`：

```javascript
const QWEN_SYSTEM_PROMPT = `你是一个专业的电力调度决策助手...
请用简洁的语言回答，每个步骤不超过50字。`;  // 调整为简洁风格
```
